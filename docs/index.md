# Lectify

## Инфраструктура

| Сервис | URL |
|--------|-----|
| **Grafana** (дашборды, метрики) | [grafana.localhost](http://grafana.localhost) |
| **Vault** (секреты) | [vault.localhost](http://vault.localhost) |
| **MinIO** (S3-хранилище) | [minio.localhost](http://minio.localhost) |
| **Prometheus** (сбор метрик) | [prometheus.localhost](http://prometheus.localhost) |
| **Loki** (логи) | [loki.localhost](http://loki.localhost) |
| **Tempo** (трейсинг) | [tempo.localhost](http://tempo.localhost) |
| **Alertmanager** (алерты) | [alertmanager.localhost](http://alertmanager.localhost) |
| **Kafka UI** (управление Kafka) | [kafka.localhost](http://kafka.localhost) |
| **Документация** | [docs.localhost](http://docs.localhost) |
