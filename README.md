# Lectify - Платформа генерации конспектов из аудиофайлов

## Структура монорепозитория

```bash
apps/               микрофронт
services/           все сервисы приложения
    | api/              api-сервисы
    | infra/            инфраструктурные сервисы
    | orchestrator/     оркестратор
    | plugins/          инфраструктура вокруг плагинов
infra/              инфраструктура приложения
proto/              контракты всех сервисов
plugins/            плагины
utils/              утилиты
libs/               shared библиотеки
docs/               документация приложения
```
